# README

此项目针对从非简并到简并的绝热演化进行的专门的探讨，预计探讨能谱和 reduced density matrix 的对比。

## 验证是否有对称性降低绝热要求

### 实验内容

选取简并点附近：$[5\times10^{-4}T,\,5\times10^{-2}T]$。

所谓降低绝热要求体现在对总时间T要求的降低。从初始状态基态出发，我们选择 $T=50$ 观察实际演化 $\psi$ 与瞬时 $H(t)$ 的本征态的 overlap 的下降程度。

为了保证演化算符分解的精确度，选择 $dt = 5\times10^{-6} \cdot T \approx 10^{-4}\,\tau$，其中 $\tau = t_{\max} - t_{\min}$ 为实际演化时间区间长度。

### 实验结果结论

演化时间 $T$ 需要满足：

$$
T \gg \max_t \frac{|\langle 1|\dot{H}|0\rangle|}{\Delta E(t)^2}
$$

如果 $\Delta E \sim 10^{-5}$(这已经是演化过程最大的量级了)，那么 $T \gg 10^{10}$ 量级。这远远大于我们实际演化时间。

### 暂时结论

应该确实跃迁被相关对称性限制，大概是：

\langle 1|\dot{H}|0\rangle = 0 \quad \text{（被对称性禁戒）}

但是我既不了解绝热演化，也不了解对称性，暂且搁置。
